<p>
 Aspose.OMR per Java è un'API per il riconoscimento ottico dei segni da varie immagini tra cui PNG, GIF, JPEG, TIFF, BMP. L'API salva il risultato nei formati CSV e JSON e può mostrare i risultati in formato testo durante l'esecuzione di operazioni OMR. Inoltre, l'API di riconoscimento OMR acquisisce i dati contrassegnati dall'uomo da file di più fonti come sondaggi, documenti MCQ, questionari e altro. L'elevata precisione viene mantenuta durante il riconoscimento delle immagini scansionate e persino delle immagini provenienti da tutte queste fonti.
</p>
