<p>
 Aspose.OMR para Java es una API para el reconocimiento óptico de marcas de varias imágenes, incluidas PNG, GIF, JPEG, TIFF y BMP. La API guarda el resultado en formatos CSV y JSON y también puede mostrar los resultados en formato de texto al realizar operaciones OMR. Además, la API de reconocimiento OMR captura datos marcados por humanos de archivos de múltiples fuentes, como encuestas, documentos MCQ, cuestionarios y más. Se mantiene una alta precisión durante el reconocimiento de imágenes escaneadas e incluso imágenes de todas estas fuentes.
</p>
