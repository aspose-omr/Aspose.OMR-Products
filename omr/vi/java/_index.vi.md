<p>
 Aspose.OMR cho Java là một API để nhận dạng nhãn hiệu quang học từ các hình ảnh khác nhau bao gồm PNG, GIF, JPEG, TIFF, BMP. API lưu kết quả ở định dạng CSV và JSON cũng như có thể hiển thị kết quả ở định dạng Văn bản khi thực hiện các hoạt động OMR. Hơn nữa, API nhận dạng OMR thu thập dữ liệu do con người đánh dấu từ các tệp của nhiều nguồn như khảo sát, bài báo MCQ, bảng câu hỏi và hơn thế nữa. Độ chính xác cao được duy trì trong quá trình nhận dạng hình ảnh được quét và thậm chí cả hình ảnh từ tất cả các nguồn này.
</p>
