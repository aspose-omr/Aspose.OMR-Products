<p>NEET는 고정된 패턴의 획일적인 시험입니다. 따라서 응시자는 교육 테스트 중에 실제 테스트 기대치를 숙지하기 위해 정확한 지침을 따라야 합니다.</p>
<ul>
	<li>최대 시험 시간은 3시간 20분(200분)입니다.</li>
	<li>질문당 하나의 거품만 채워야 합니다.</li>
	<li>거품의 원은 펜이나 ​​검은색 마커로 완전히 채워야 합니다.</li>
	<li>특히 거품이나 바코드를 가로지르는 흔적을 피하십시오.</li>
	<li>학생의 이름과 연락처 정보를 입력하려면 각 열에 문자 또는 숫자를 사용하여 말풍선을 표시하십시오.</li>
</ul>

<h3>NEET 시트 커스터마이징</h3>

<p>NEET 시험은 물리학, 화학, 식물학, 동물학 섹션으로 나누어진 180개의 질문으로 통일된 형식을 가지고 있습니다. 교육을 위해 원래 패턴을 따르는 것이 매우 권장되지만 섹션을 제거하거나 해당 섹션의 답변 수를 변경하여 테스트의 특정 영역에 집중하도록 양식을 쉽게 조정할 수 있습니다.</p>

<p><b>니트 시트 사용자 정의:</b></p>

<ul type="1">
	<li>일반 텍스트 편집기로 NEET 양식의 소스 코드(neet.txt)를 엽니다.</li>
	<li>지침에 따라 소스 코드를 변경합니다.</li>
	<li>용지 크기, 거품 색상, 글꼴 및 기타 전체 페이지 설정을 사용자 정의합니다.</li>
</ul>
