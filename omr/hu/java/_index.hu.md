<p>
 Az Aspose.OMR for Java egy API a különféle képek optikai jeleinek felismerésére, beleértve a PNG, GIF, JPEG, TIFF, BMP fájlokat. Az API elmenti az eredményt CSV és JSON formátumban, valamint szöveges formátumban is megjelenítheti az eredményeket az OMR műveletek végrehajtása során. Ezen túlmenően, az OMR felismerő API az ember által megjelölt adatokat több forrásból származó fájlokból rögzíti, például felmérésekből, MCQ-papírokból, kérdőívekből és egyebekből. A nagy pontosság megmarad a beolvasott képek, sőt az összes ilyen forrásból származó képek felismerése során.
</p>
