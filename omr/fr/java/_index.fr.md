<p>
 Aspose.OMR pour Java est une API pour la reconnaissance de marques optiques à partir de diverses images, notamment PNG, GIF, JPEG, TIFF, BMP. L'API enregistre le résultat aux formats CSV et JSON et peut afficher les résultats au format texte lors de l'exécution d'opérations OMR. De plus, l'API de reconnaissance OMR capture des données marquées par l'homme à partir de fichiers de plusieurs sources telles que des enquêtes, des documents QCM, des questionnaires, etc. Une grande précision est maintenue lors de la reconnaissance des images numérisées et même des images provenant de toutes ces sources.
</p>
