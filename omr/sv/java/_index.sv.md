<p>
 Aspose.OMR för Java är ett API för igenkänning av optiska märken från olika bilder inklusive PNG, GIF, JPEG, TIFF, BMP. API sparar resultatet i CSV- och JSON-format samt kan visa resultaten i textformat när du utför OMR-operationer. Dessutom fångar OMR recognition API mänskligt märkt data från filer från flera källor såsom undersökningar, MCQ-papper, frågeformulär och mer. Hög noggrannhet bibehålls under igenkänning av skannade bilder och även bilder från alla dessa källor.
</p>
