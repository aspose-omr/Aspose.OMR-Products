<p>
 Aspose.OMR для Java — это API для распознавания оптических меток из различных изображений, включая PNG, GIF, JPEG, TIFF, BMP. API сохраняет результат в форматах CSV и JSON, а также может отображать результаты в текстовом формате при выполнении операций OMR. Кроме того, API-интерфейс распознавания OMR собирает отмеченные человеком данные из файлов из нескольких источников, таких как опросы, документы MCQ, анкеты и многое другое. Высокая точность сохраняется при распознавании отсканированных изображений и даже картинок из всех этих источников.
</p>
