 Aspose.OMR for Java 是一個用於從各種圖像（包括 PNG、GIF、JPEG、TIFF、BMP）中識別光學標記的 API。 API 以 CSV 和 JSON 格式保存結果，並且可以在執行 OMR 操作時以文本格式顯示結果。此外，OMR 識別 API 從調查、MCQ 論文、問卷等多個來源的文件中捕獲人工標記數據。在識別所有這些來源的掃描圖像甚至圖片期間，保持高精度。
