<p>
 Aspose.OMR para Java é uma API para reconhecimento de marcas ópticas de várias imagens, incluindo PNG, GIF, JPEG, TIFF, BMP. A API salva o resultado nos formatos CSV e JSON, bem como pode mostrar os resultados em formato de texto na execução de operações OMR. Além disso, a API de reconhecimento OMR captura dados marcados por humanos de arquivos de várias fontes, como pesquisas, documentos MCQ, questionários e muito mais. A alta precisão é mantida durante o reconhecimento de imagens digitalizadas e até imagens de todas essas fontes.
</p>
