<p>Η βιβλιοθήκη μας υποστηρίζει την πλήρη ροή εργασιών OMR - από το σχεδιασμό μιας φόρμας έως την αναγνώριση των συμπληρωμένων έντυπων αντιγράφων της. Μπορείτε να συνδυάσετε έναν μεγάλο αριθμό προκαθορισμένων στοιχείων σε φόρμες που ταιριάζουν καλύτερα στους στόχους και το κοινό-στόχο σας και να εξατομικεύσετε και να επωνυμίσετε περαιτέρω τις φόρμες OMR σας προσθέτοντας λογότυπα, γραμμικούς κώδικες, κωδικούς QR ή εικόνες. Οι φόρμες περιγράφονται σε απλό κείμενο ή συμβολισμό JSON ή απευθείας στον κώδικα του προγράμματος. δεν απαιτούνται εργαλεία σχεδιασμού. Οι ερωτηθέντες μπορούν να συμπληρώσουν φόρμες με στυλό, μολύβι ή μαρκαδόρο και να χρησιμοποιήσουν οποιοδήποτε είδος σήμανσης - η τεχνητή νοημοσύνη και η προηγμένη βαθμονόμηση ακρίβειας εγγυώνται αξιόπιστα αποτελέσματα αναγνώρισης ακόμη και χωρίς εξειδικευμένους σαρωτές.</p>
